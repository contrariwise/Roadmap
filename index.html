Webbutveckling

  • 1

    HTML och CSS

    Med HTML och CSS kan man göra en såkallad "statik" webbsida, dvs en webbsida som står stilla med bilder och text. Ingen interaktion eller funktioner. Boken HTML och CSS boken är den enda jag kan hitta på svenska som är modern. Jag har inte läst denna bok, så jag kan inte stå upp för kvaliteten. Om du hittar någon annan bok i HTML och CSS, se till att den är HTML5 och CSS3, annars är den för gammal. Om man är duktig i engelska, Head First HTML & CSS är mycket bra.

  • 2

    Programmeringsbegrepp

    Man ska lära sig att programmera, inte särskilt att programmera på webben, bara teori på hur programmering funkar. Jag rekommenderar starkt att man lär sig med språket Python för det är mycket enkelt för nybörjare.

    Språket Java (alltså inte Javascript) funkar också. Jag rekommenderar inte att man lär sig webbprogrammeringspråken PHP eller Javascript innan man har klarat denna steg, eftersom de har dålig struktur och man lär sig dåliga vanor om man inte vet hur man gör.

    Denna steg är inte spännande för alla. Denna är ganska seriösa studier, och liknar att plugga matematik. Om man vill bara hobby-programmera, hoppa över Steg 2 och 3. Programmering i Python är den enda jag kan hitta på svenska. Denna har jag inte läst. Om man är duktig i engelska, Python Programming for the Absolute Beginner är väldigt, väldigt bra. Denna var boken jag började med själv för några år sen.

  • 3

    Objektorienterad

    Mer avancerad, men om allting går bra så länge, den nästa man kan lär sig är objektorienterad programmering. Då måste man stå ut med engelska. Head First Java är underbar.

  • 4

    PHP

    Nu kan man skapa interaktiva webbsidor! Man har redan lärt sig HTML och CSS, men nu kan man skriva logik som genererar HTML och CSS. PHP är kod som körs på servern innan den serveras till webbläsaren. Den kallas för "backend programmering." Programming PHP är den bästa, och på svenska finns det Webbutveckling med PHP och MySQL.

  • 5

    Javascript

    Javascript är den andra sidan till interaktiva webbsidor. Javascript körs i webbläsaren, vilken är "frontend programmering." Om man kan ta det på engelska, Head First JavaScript Programming är en bra introduktion. Denna bok går bra att sätta igång med Javascript, och funkar även om den är mer hobby än seriös och man har hoppat över Steg 2 och 3. Javascript är en svensk bok som jag inte har läst. Om man är mycket seriös, JavaScript: The Good Parts måste alla professionella webbutvecklare läser.

  • 6

    Experiment

    Nu är man en webbutvecklare och man kan experimentera med vilken språk eller framework som helst! Ruby on Rails, Meteor, Go, Backbone, Node, vad som helst. Lycka till!
